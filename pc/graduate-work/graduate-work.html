О работе

Выпускная квалификационная работа бакалавриата по направлению «09.03.01 — Информатика и вычислительная техника» включает в себя 4 раздела:

  • Исследовательский — исследование предметной области
  • Аналитический — проектирование системы
  • Технологический — реализация сисетмы
  • Экономический — планирование и оценка стоимости

Работа посвящена разработке информационной системы управления общественным транспортом. ИС состоит из 3-x подсистем (двух клиентских и одной серверной):

  • Проектирования (TM.Editor)
  • Управления (TM.Studio)
  • Развертывания (TM.Server)

Подробнее об архитектуре системы и ее разработке вы можете прочитать в файле ВКР. В рамках дипломной работы реализуется только часть функционала подсистем управления и развертывания.

Инормационная система разрабатвается на языке программирования C# (платформа .NET Core 7.0). Для реализации клиентских подсистем ИС используется платформа WPF. Серверная часть реализуется при помощи ASP.NET WebApi. Взаимодействие с базой данных производится при помощи фреймворка EntityFramework.
