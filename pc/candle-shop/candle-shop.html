О проекте

Данные проект реализовывался в рамках практики по программе цифровой кафедры. Сайт-визитка представляет собой сайт с небольшим количеством страниц. Его назначение предоставить потенциальным покупателям информацию о тварах.

Это приложение создано при помощи ASP.NET веб-API (платформа .NET 7.0) на языке программирования C#. База данных реализована при помощи ORM Entity Framework. В качестве сервера базы данных используетя MS SQL Server. Клиентская часть приложения реализована с помощью библиотке React на языке программирования TypeScript.
