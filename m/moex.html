О проекте

iMOEX — мобильное приложение для операционной системы Android, которое позволяет следить за новостями и событиями Московской Биржи, а также просматривать ценные бумаги и их статистику.

Приложение создано с использованием фреймворка Xamarin.Forms. Логика приложения реализуется на языке C#. В качестве API используется ИСС Московской Биржи.
