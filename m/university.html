О проекте

Цель данного проекта — показать какими могли бы быть сайты высших учебных заведений России. Интуитивная и красивая клиентская часть в сочетании с функциональной сервреной — две составляющие идеального сайта.

Это веб-приложение создано при помощи форм ASP.NET WebForms (.NET Framework 4.8) на языке программирования C#. База данных реализована с помощью Entity Framework, а также языка запросов T-SQL. В качестве сервера базы данных используетя MS SQL Server.
