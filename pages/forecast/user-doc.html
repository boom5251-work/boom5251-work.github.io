WeatherForecast — документация пользователя

О приложении

Приложение «WeatherForecast» — прогноз погоды — одностраничное веб-приложение, предназначенное для получения большого спектра метеорологических данных, включая текущие погодные условия, прогноз погоды на сутки, астрономические данные и т.д.

Геолокация и поиск

Для просмотра метеорологических данных необходимо предоставить доступ к геолокации. После этого актуальные данные будут отображены.

Есть и другой способ посмотреть прогноз. Можно ввести название населенного пункта вручную (например: /Moscow). Также можно вручную ввести координаты, для которых нужно получить метеоданные (например: /55.883522,37.646053).

Полученные метеоданные

В левом углу загруженной страницы можно увидеть местоположение, регион и страну. В центре находится иконка, отображающая текущие погодные условия, а также температуру воздуха. В правом углу находятся все самые важные метеоданные. Внизу расположены общие данные прогноза погоды на сутки.
