WeatherForecast — техническая документация

Контроллеры

WeatherController Контроллер обрабатывает поступающие на сервер GET-запросы. Возвращает клиенту страницу с метеоданными.

Модели представления

IConditionViewModel Интерфейс модели представления. Содержит основные погодные данные.
AstronomyViewModel Модель представления. Содержит астрономические данные.
ForecastViewModel Модель представления. Содержит прогноз погоды на сутки.
CurrentWeatherViewModel Модель представления. Содержит текущие погодные данные.
InfoViewModel Модель представления. Содержит конфигурационные данные страницы.
ForecastViewModelsExtension Представляет методы расширения для списка прогнозов (прогноза на сутки).
WeatherViewModel Модель представления. Содержит все погодные данные.

Получение и обработка данных

Modeler Модуль создает и инициализирует все модели представления на основе данных, полученных с сервера метеоданных WeatherApi.
WeatherApi Модуль отвечает за отправку запросов на сервер метеоданных WeatherApi и получение ответов в формате JSON.

Зависимости

IConditionIconService Интерфейс зависимости, предназначенной для получения иконок погодных состояний.
ConditionIconService Класс зависимости, предназначенной для получения иконок погодных состояний.
