Modeler Модуль

Определение

Пространство имен: WeatherForecast.Data

Сборка: WeatherForecast.exe

Модуль создает и инициализирует все модели представления на основе данных, полученных с сервера метеоданных WeatherApi.

module public Modeler =

dateTimePattern

Строка-паттерн даты и времени.

F# 
let private dateTimePattern: string = "yyyy-MM-dd H:mm"

timePattern

Строка-паттерн времени.

F# 
let private timePattern: string = "h:mm tt"
